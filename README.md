# ESDL
Data Lake by ES

# 1. Usage
## 1.1 Set some env

#### Highly dependent env.

|   env_key |   description | default |
|   :---:    |   :---:    |   :---:    |
|   DL_PORT    |   开启http端口   |  9202
|   ES_HOST    |   连接到ES到主机ip    |  127.0.0.1
|   ES_PORT    |   连接到ES到主机port    |  9200
|   BP_LOG_OUTPUT    |   日志存放路径   |  console
|   BP_LOG_LEVEL    |   日志存放等级   |  info
