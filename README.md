# ESDL
Data Lake by ES

# 1. Usage
## 1.1 Set some env

#### Highly dependent env.

|   env_key |   description | default |
|   :---:    |   :---:    |   :---:    |
|   DL_PORT    |   开启到监听端口   |  9201
|   ES_HOST    |   连接到ES到主机ip    |  127.0.0.1
|   ES_PORT    |   连接到ES到主机port    |  9200
|   LOG_PATH    |   日志存放路径   |  dl.log
